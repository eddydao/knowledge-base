# Document Object Model (DOM)
- HTML pages is like a tree structures
- DOM is an API that enables modification of the element trees corresponding to the web pages

# Manipulate the DOM element
- the top node of the DOM tree called *document* object
- access the element in DOM tree by using the code
```
elem = document.getElementByTagName(’’);
elem = document.getElementById(‘’);
```
