# Spring Proxy
Spring will create dynamic Cglib proxies, proxy classes through subclassing them on the fly 
Giving the beans additional features, without modifying the code => Aspect oriented programming

# AOP
A programming paradigm that allows to modularize crosscutting concerns, that id, add additional features, without modifying the actual code in the class/ method
- Encapsulate code into separate, reusable modules called aspects
- These aspect can be applied to specific parts of the application

> Wrap the code around with aspect code and add features to it without changing the code
