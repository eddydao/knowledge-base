# Project Requirement Docs for Ticket.me
## 1. Project overview
Ticket.me is a revolutionary decentralized event ticket web-based application where Event organizers can create events and mint NFT-based tickets. Buyers can purchase, store, transfer tickets securely in their wallets. Staff can validate tickets at the event via a QR-code scanner tied to on-chain data.
## 2. Key features
Ticket.me provides ability of managing event details on the platform, including create, update, close, delete event and event information for Event creator and manager. The platform provides NFT (ERC-721) tickets minted on the blockchain for ticket minting. Tickets of event can be transfer between wallets with transactions recorded on-chain for transparency. Also when the event start, staffs can easily verify ticket ownership at the event entry with QR-code scanner on their mobile app.

## 3. Tech stack
Ticket.me using Polygon for blockchain technology, use OpenZeppelin libraries (ERC-721) for smart contract development.
The frontend will be using React.js and React Native for building website and mobile version. The Backend using Node.js and Express for API routes. The platform will use PostgreSQL for centralized database to store event information off-chain. Event staff will use mobile version with QR-code scanner with html5-qrcode library.
